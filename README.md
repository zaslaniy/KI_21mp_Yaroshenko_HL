Create docker-compose.yaml for container deploying.:
Створюємо в папці C:\Users\User\Documents\5 курс\1 сем\HL\lab_1 файл docker-compose.yaml

- `cd C:\Users\User\Documents\5 курс\1 сем\HL\lab_1`
